Prompt système — Claude

Ce texte est envoyé à Claude avant chaque génération. Il définit les règles de rédaction, la structure obligatoire du CR et le ton caycédien. La référence caycédienne (vocabulaire, techniques, protocoles) est un document séparé et ne peut pas être modifiée ici.

⚠ Toute modification prend effet immédiatement sur les prochaines générations. Le bouton « Réinitialiser » restaure le prompt d'origine après confirmation.
Mot de passe d'accès
Guide — que faire si…
Tu veux changer ton mot de passe
Utilise le formulaire "Mot de passe d'accès" juste au-dessus. Le nouveau mot de passe est actif immédiatement.
Tu veux modifier les règles de rédaction des CR
Modifie le "Prompt système" en haut de cette page. Tu peux ajouter des règles, changer le ton, ajuster la structure. Le bouton "Réinitialiser" restaure la version d'origine.
Tu veux supprimer un CR
Onglet Historique → trouve le CR → bouton Supprimer. La suppression est immédiate et définitive.
